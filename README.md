* The compilers that do not work in my laptop
    * MSVC2015 32bit
    * MSVC2015 64bit
    * UWP 64bit (MSVC 2015)
    * uwp armV7 (MSVC 2015)
  
* The compilers that do work in my laptop
    * MinGW 32-bit
    * MinGW 64-bit

* When you build OpenCV using CMake, the directory of the source code is `.../opencv/source`, not `.../opencv`
